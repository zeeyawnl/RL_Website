About The Game

Gameplay

RL algorithms allow game characters (agents) to explore the game environment, make decisions, and receive rewards like winning outcomes or penalties.

Motivation

The motivation behind using reinforcement learning in gaming appeals to users by highlighting the exciting possibilities of developing intelligent and adaptive AI agents capable of mastering complex environments.

Enemies

Games provide the perfect testing ground because they simulate dynamic, rule-based worlds where agents can experiment, learn from feedback, and adapt strategies based on their performance.

Environment

The environment provides real-time feedback to the agents by sending state information (game status, player position, score) and receiving actions from agent.
